    How to Upgrade from Fedora 38 to Fedora 39

    升级Fedora39前准备

    在升级Fedora39之前务必做好以下准备：

    1. 备份数据。
    2. 确保电源稳定。
    3. 确保网络正常。

    升级Fedora39步骤

    1. 更新Fedora38 将Fedora38的所有软件包升级到最新版本 
      sudo dnf upgrade --refresh
    2. 安装dnf-plugin-system-upgrade 一般情况下dnf-plugin-system-upgrade已经安装了，如果未安装可以执行以下命令安装 
      sudo dnf install dnf-plugin-system-upgrade -y
    3. 升级到Fedora39 执行以下命令升级到Fedora39 
      sudo dnf system-upgrade download --releasever=39
      如果执行上个命令中出现包不兼容，导致升级失败的情况下，可以使用以下命令 
      sudo dnf system-upgrade download --releasever=39 --allowerasing.
      上个命令执行成功的情况下执行以下命令升级 
      sudo dnf system-upgrade reboot
      系统重启后，开始执行升级。大约30分钟左右即可升级完成。

    升级Fedora39成功后的操作

    确认升级Fedora39成功之后执行以下操作删除用不到的包和linux内核

    sudo dnf autoremove -y
sudo dnf remove --oldinstallonly -y

    上述操作执行完之后，你将得到一个全新的升级成功的Fedora39系统。
