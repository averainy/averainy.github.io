    Wireguard Install Script

    最近用wireguard安装脚本搭建wireguard服务器，用起来非常方便，推荐使用。
    在下载脚本之后，强烈建议修改下生成客户端脚本的代码第313行，加入以下配置：

    PersistentKeepalive = 25

    这样每隔25秒就会发送一次心跳包，以保持客户端与服务端的连接。

    如果不希望所有流量都走wireguard服务端，建议修改第314行代码：

    AllowedIPs = 10.66.66.0/24" >>"${HOME_DIR}/${SERVER_WG_NIC}-client-${CLIENT_NAME}.conf"

    这样只有vpn网段的流量才会通过vpn服务端中转。
