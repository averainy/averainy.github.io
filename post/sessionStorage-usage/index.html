    sessionStorage用法

    sessionStorage存储的数据会在会话结束（如关闭标签页或者浏览器时）的时候清除掉。每新开一个新的标签页就会有新的sessionStorage。各个sessionStorage之间无法互相访问。

    sessionStorage语法

    // 保存到sessionStorage
sessionStorage.setItem('key', 'value');
// 从sessionStorage取数据
sessionStorage。getItem('key')
// 从 sessionStorage 删除保存的数
sessionStorage.removeItem('key')
// 从 sessionStorage 删除所有保存的数据
sessionStorage.clear();
